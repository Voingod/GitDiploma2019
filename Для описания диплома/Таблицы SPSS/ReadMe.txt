1. ForDisertation - таблица с данными для регрессионного анализа (чтобы понять какие кластеры похожы)
2. Для кластеров - таблица для загрузки в программу, чтобы определить количество класстеров
3. Character - таблица с описание харакетристик для 7 кластеров мужчин и 8 кластеров женщин
4. Character5 - таблица с описание харакетристик для 5 кластеров мужчин и 5 кластеров женщин
6. ResultTable - таблица, содержащая средние (центроиды) значения АТС, АТД, ЧСС в состоянии покоя и на каждой
минуте после нагрузки для каждого кластера отдельно (7 кластеров для мужчин, 8 - для женщин)
7.ResultTable5 - таблица, содержащая средние (центроиды) значения АТС, АТД, ЧСС в состоянии покоя и на каждой
минуте после нагрузки для каждого кластера отдельно (5 кластеров для мужчин, 5 - для женщин)
8.повторные бойко1 (Настенко исходник) - таблица с исходными данными, которые потом используются в таблицах
"ForDisertation" и "Для кластеров"
9. Таблицы с уменшением класстеов/M2B_poly_Ul_Chart - содержит графики для мужчин при 5,4,3 количествах 
кластеров, а также данные для их построения (центроиды кластеров)
10. Таблицы с уменшением класстеов/F3B_poly_Ul_Chart - содержит графики для женщин при 5,4,3 количествах 
кластеров, а также данные для их построения (центроиды кластеров)
