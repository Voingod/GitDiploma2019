1. ForDisertation - таблица с данными для регрессионного анализа (чтобы понять какие кластеры похожы)
2. Для кластеров - таблица для загрузки в программу, чтобы определить количество класстеров
3. Character - таблица с описание харакетристик для 7 кластеров мужчин и 8 кластеров женщин
4. Character5 - таблица с описание харакетристик для 5 кластеров мужчин и 5 кластеров женщин
5. ResultTable - таблица, содержащая средние (центроиды) значения АТС, АТД, ЧСС в состоянии покоя и на каждой
минуте после нагрузки для каждого кластера отдельно (7 кластеров для мужчин, 8 - для женщин)
6.ResultTable5 - таблица, содержащая средние (центроиды) значения АТС, АТД, ЧСС в состоянии покоя и на каждой
минуте после нагрузки для каждого кластера отдельно (5 кластеров для мужчин, 5 - для женщин)
